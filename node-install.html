Ubuntu Nodejs install

Lyndon NodejsStudy

在ubuntu下用apt安装nodejs太坑，现在记录下另外两种安装方式以备用

二进制包安装

  1. 官网下载二进制包,并解压到相应目录

    $ wget https://nodejs.org/dist/v4.2.3/node-v4.2.3-linux-x64.tar.gz
$ tar xvzf node-v4.2.3-linux-x64.tar.gz
    1. 设置环境变量 ```bash $ vim ~/.bashrc

    #set nodejs env export NODE_HOME=“the Node path” export PATH=$NODE_HOME/bin:$PATH export NODE_PATH=$NODE_HOME/lib/node_modules:$PATH ```

  2. 设置生效

    $ source ~/.bashrc
    1. 设置npm国内镜像 bash $ npm config set registry http://registry.cnpmjs.org

源码安装

  1. 官网下载源码包,并解压到相应目录

    $ wget https://nodejs.org/dist/v4.2.3/node-v4.2.3.tar.gz
$ tar xvzf node-v4.2.3.tar.gz
    1. 进入目录进行编译 bash $ ./configure --prefix=/usr/local/nodejs $ make && make install

  2. 设置环境变量

    $ vim ~/.bashrc
#set nodejs env
export NODE_HOME="the Node path"
export PATH=$NODE_HOME/bin:$PATH
export NODE_PATH=$NODE_HOME/lib/node_modules:$PATH
    1. 设置生效 bash $ source ~/.bashrc

  3. 设置npm国内镜像

    $ npm config set registry http://registry.cnpmjs.org
